# Jaktopplevelsen Premium

> Ingen AI-tull. Bare verktøyene en jeger faktisk trenger.

## Gratis (Kjernen)
- Loggføring av turer, vær, observasjoner, fellinger
- Garmin GPX-import
- Hundestatistikk
- Enkelt bildegalleri

---

## Jeger Pro — kr 149/år
For solojegeren.

| Funksjon | Beskrivelse |
| :--- | :--- |
| **Offline-modus** | Logg uten dekning, synk senere |
| **Full eksport** | PDF, CSV, GPX. Din data er din |
| **Spor-overlay** | Se alle spor i samme område over tid |
| **Sesongrapporter** | Auto-generert PDF med statistikk |
| **Hundesammenligning** | Ytelse mellom hunder over tid |

---

## Jaktlag — kr 99/bruker/år (min. 3)
For laget.

| Funksjon | Beskrivelse |
| :--- | :--- |
| **Felles logg** | Ett sted for lagets turer |
| **Digital postplan** | Tegn poster på kart, del med alle |
| **Live posisjon** | Se alle i laget under jakta |
| **Delt galleri** | Alle bilder samlet |
| **Vaktliste** | Kalender for jaktdager |

---

## Terreng Pro — kr 499/år
For terrengforvaltere (tillegg).

| Funksjon | Beskrivelse |
| :--- | :--- |
| **Kvoter & felling** | Sett kvoter, se gjenstående |
| **Observasjonskart** | Heatmap over viltobservasjoner |
| **Hjorteviltregister-eksport** | Riktig format |
| **Gjestejakt** | Invitasjonslenker for gjester |
